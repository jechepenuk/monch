APACHE REPORT
Apache is an open-source web server software. Simply put, Apache is an http web server that serves and delivers web content to visitors. It was originally created by Robert McCool and is written in the C language. Apache works by accepting requests and passing it through a set of modules. These modules correspond to different types of functionality such as security. Apache is a combination of a multiprocess and multi-threaded model whereby each process has control over the number of threads.It is a cross-platform web server and works on operating systems such as Microsoft   Windows  and Linux. It is an industry standard because it is well known, reliable, and open-source/free.
